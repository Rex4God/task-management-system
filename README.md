## Setting up for Development

- Clone project.
- Install Node ( >= 14.0.0).
- Install yarn/npm.
- Copy .env.example to .env file
- Run yarn/npm install.
- Run yarn/npm start.

## Execute
- yarn/npm start 
- yarn/npm run dev
